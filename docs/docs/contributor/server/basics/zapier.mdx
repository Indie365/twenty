## Setup
- Ask an admin to access the **Twenty 1Password** account
- Install [Zapier CLI](https://platform.zapier.com/quickstart/cli-tutorial) globally: `npm install -g zapier-platform-cli`
- Login with CLI: `zapier login`
  > email: shared@twenty.com

  > password: Find it in the **Vault TwentyShared**, look for Zapier credentials

- Install Zapier packages:
```
cd packages/twenty-zapier
yarn
```
- set environment variables:
  - In **packages/twenty-zapier** launch `cp .env.example .env`
  - [launch backend application](/contributor/server/basics/workflows#starting-a-new-feature)
  and go to `http://localhost:3000/graphql`
  - set **Headers** with `{"Authorization": "Bearer YOUR_TOKEN"}`.
  Find your token by connecting to the
  [local front application](/contributor/frontend/basics/contributing#starting-a-new-feature)
  and find the token in **graphql** requests' header.
  - generate an apiKey:
  ```
  mutation createApiKey {
    createOneApiKey(data:{name:"Test", expiresAt: "2100-01-01 10:10:10.000"}){
      token
      expiresAt
    }
  }
  ```
  - copy the generated api key and paste it to replace the .env **YOUR_API_KEY** value
- `cd .. && cd twenty-zapier` to set environment variables (needs autoenv)
- Test: `zapier test`
- List all Zapier CLI commands: `zapier`
