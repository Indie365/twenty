## When to use

Use checkboxes to: 

- Allow users to select one or more options from a list of options or to indicate whether they agree to the terms & conditions  
- Get the user's explicit confirmation to apply certain settings
- Filter data and for batch editing 

## When not to use 

Avoid using checkboxes when:

- You want to allow users to turn an option on or off. Use a toggle instead.
- Users can only select one option from a list. Use radio buttons instead.
